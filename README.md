# GitBook LaTeX theme

This Gitbook theme uses [Typeset](https://github.com/bramstein/typeset) as an implementation of the Knuth and Plass line breaking algorithm. Uses [Hypher](https://github.com/bramstein/hypher) for hyphenation.
